## About
Just a pizza lover runnin' for a pizza.

## Dependencies and installation
### dependencies
Requirements:
<br/>SDL2

On arch:
<br/>sudo pacman -s sdl2

On ubuntu:
<br/>sudo apt-get install libsdl2-dev

Build:
<br/> build by running 'make' in root of the project
